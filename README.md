# Ruta Viva

Landing y demo MVP para el hackaton Impulso Comercio Canarias 2026.

## Desarrollo

```bash
npm install
npm run dev
```

La app queda disponible en `http://localhost:5173/` por defecto.

## Verificacion

```bash
npm run lint
npm run build
```

## Alcance demo

- Landing de una pagina.
- Pasaporte digital simulado con `localStorage`.
- Recomendador determinista desde `src/data/businesses.mock.json`.
- Mapa esquematico sin dependencia externa.
- Mini dashboard con metricas sinteticas.
